## Apercu

Site one-page responsive presentant mon parcours, mes realisations et mes coordonnees. Optimise pour la performance et l'accessibilite.

## Tech Stack

- **HTML5** - Structure semantique
- **CSS3** - Flexbox, Grid, variables CSS
- **GitHub Pages** - Hebergement
- **Google Fonts** - Typographie (Inter)
