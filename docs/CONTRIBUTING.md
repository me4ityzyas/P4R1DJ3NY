# Contributing

Les contributions sont les bienvenues. Voici comment participer.

## Comment contribuer

1. Forkez le repository
2. Creez une branche (`git checkout -b feature/amelioration`)
3. Commitez vos changements (`git commit -m 'Ajout amelioration'`)
4. Pushez (`git push origin feature/amelioration`)
5. Ouvrez une Pull Request

## Regles

- Code propre et lisible
- Pas de framework, on reste en vanilla HTML/CSS
- Testez le responsive avant de soumettre
- Images optimisees (max 200KB)

## Bugs

Ouvrez une issue avec :
- Description du probleme
- Navigateur et OS utilises
- Captures d'ecran si necessaire
